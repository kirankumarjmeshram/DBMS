1757. Recyclable and Low Fat Products
Write a solution to find the ids of products that are both low fat and recyclable.
Table: Products
+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| product_id  | int     |
| low_fats    | enum    |
| recyclable  | enum    |
+-------------+---------+

sol: SELECT product_id from Products WHERE low_fats = 'Y' AND recyclable = 'Y'
